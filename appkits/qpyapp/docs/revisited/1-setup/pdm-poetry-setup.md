# PDM Project Setup

How to use pdm setup python project:
1. init
2. add dependencies
3. build
4. scripts like npm scripts

## Init Project
```sh
pdm init
```

## Add/Install dependencies


```sh
pdm add 
pdm install

```
## Build Project

```sh
pdm build
```

## pdm scripts

How to setup pdm scripts.

```shell
pdm run cleanup
```
